Environments
Files which configure the overall game play of a particular server
File Description
beta.xml Empty -- used by Overcast on the beta testing server
development.xml Empty -- used by Overcast on development realm servers
global.xml Main configuration file for servers -- increases hitbox size and reverts combat to 1.8 stats
production.xml Empty -- used by Overcast for non-development realm servers
ranked.xml Empty -- used by Overcast for ranked realm servers
tournament.xml Used by Overcast for tournament realm servers
unranked.xml Empty -- used by Overcast for non-ranked / non-tournament realm servers
