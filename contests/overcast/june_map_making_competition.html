    Details and Guidelines
    Only CTF and KoTF maps were allowed, allowing for 10v10 gameplay
    Maps being created were to be designed for tournament environments
    Upon the approval of an Event Coordinator, select contest maps would be included in Overcast's 10th tournament
    XML documents were to be included with the submission

    First Map residence, winner trophy, and a donor rank
    Second Map residence, and a donor rank
